# ---NBS
Το έργο φέρνει τα νήπια σε επαφή με τη φύση μέσω του ρομπότ Photon, που καθοδηγεί σε λαχανόκηπο, κομποστοποιητή, ξενοδοχείο ζουζουνιών και άλλα. Με QR codes και micro:bit μαθαίνουν να παρατηρούν, συνεργάζονται και παίρνουν αποφάσεις, ενισχύοντας δεξιότητες ζωής, STEM και οικολογική συνείδηση μέσα από βιωματική μάθηση.
